# Java大杂烩
> 目前项目分为五大模块：
> - chaos-algorithm 算法与数据结构
> - chaos-common 工具类与JAVA基础
> - chaos-frame 框架
> - chaos-mode 设计模式
> - chaos-web 网关

## chaos-frame 框架
- redis NoSQL
- spring-task 定时任务
- aop 面向切面编程
- lambda 函数式编程

## chaos-mode 设计模式
- 策略模式 strategy
- 工厂模式 factory
- 工厂方法模式 factory2method
- 装饰者模式 decorator
- 代理模式 proxyPerson
- 原型模式 prototype
